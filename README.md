# NLP Frontend App – Nowf Salim

This is a Natural Language Processing (NLP) frontend project built with JavaScript and bundled using Webpack. It likely supports features like user input, processing, and output display through a web interface.

# Project Features

- Built using JavaScript ES6+
- Webpack for module bundling
- Service worker for offline support
- Ready for production and development builds

# Project Structure
├── dist/ # Bundled app (output)
├── node_modules/ # Project dependencies
├── webpack.dev.js # Development config
├── webpack.prod.js # Production config
├── package.json # Project metadata & dependencies
├── .babelrc # Babel configuration
├── .gitignore
└── README 


# How to Run
- Build for production
npm run build-prod

- development server
npm run start

# Dependencies
- Webpack
- Babel
- Node.js
